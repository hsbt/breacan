Breacan
=======

![breacan](http://www.lindaclifford.com/Images/CorkGreatKiltA.jpg)

Simple Ruby wrapper for Slack API.

Installation
------------

Add this line to your application's Gemfile:

```ruby
gem 'breacan'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install breacan
